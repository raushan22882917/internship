Upload a PDF



Instructions:

  1. Select a PDF file from your device by clicking the "Choose File" button.
  2. Ensure the file selected is in PDF format.
  3. Click the "Upload" button to start the process.
  4. The system will extract all images from the uploaded PDF and create a zip file containing these images.
  5. Once the process is complete, you will be redirected to download the zip file.
  6. Click the download link to save the zip file containing the extracted images to your device.
